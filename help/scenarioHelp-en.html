Scenarios


In this tab, user can manage launches for each scenario / specie tuple.
For each tuple, user must launch following steps (in this order) :
- Landuse : generate landuse layer (reclassified and merged)
- Friction : generate friction layer
- Project : creates Graphab project with patches
- Linkset : creates Graphab linkset (paths between patches)
- Graph : creates Graphab graph

Once graph is computed, user can press Spatialize button to compute local metric on each node.

Finally, Compare scenarios button allows user to compute global metric on each scenario / specie selected and to display results in a plot.
