Launches


In this tab, user can manage launches for each scenario / specie /extent tuple.
For each tuple, user must launch following steps (in this order) :
1 - Landuse : generate landuse layer (reclassified and merged)
2 - Friction : generate friction layer
3 - Project : creates Graphab project with patches
4 - Linkset : creates Graphab linkset (paths between patches)
5 - Graph : creates Graphab graph

Once dispersal capacity is computed, user can compute potential Dispersal areas.

Once graph is computed, user can press Local metric button to compute local metric on each node.

Finally, 7 - Compare scenarios button allows user to compute global metric on each scenario / specie selected and to display results in a plot.


Option Delete existing results ensure current and following step results are erased.

Option Load base layers loads results layers for step 1 and 2.

Option Links < X * max disp enables user to lighten linkset step considering only links with cost < X * max disp (specie disperal capacity). Use this option with cost distance unit for specie dispersal capacity.
