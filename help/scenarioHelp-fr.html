Scenarios


Cet onglet permet de définir des scenarios / variantes.
Il existe par défaut un scenario initial qu'il est possible de renommer mais qu'il ne faut pas supprimer.
Il faut ensuite créer de nouveaux scenarios par empilement en appuyant sur le bouton Ajouter un scenario et en renseignant le scenario de base ainsi que la couche de modification d'occupation du sol. Cette couche, qui peut être au format vecteur ou raster, peut contenir les valeurs numériques qui correspondent aux nouveaux postes d'occupation du sol (cf onglet de friction) ou alors être entièrement reclassifiée vers une valeur fixe. Les règles de reclassification sont affichées dans le tableau Classes de l'onglet Données.
L'option Utiliser l'emprise de la couche détermine si la couche renseignée correspond à l'emprise projet autour de laquelle la modélisation va s'effectuer. Il faut par exemple cocher cette option pour un tracé d'infrastructure linéaire mais cela n'est pas nécessaire pour les aménagements type passage à faune.
L'option Créer un scenario long terme permet de dupliquer le scenario dans le cas d'une mesure de compensation dont le niveau de fonctionnalité n'est pas le même à court et long terme. Le scenario long terme reprend le nom du scenario long terme en lui ajoutant le préfix Long. Il faut s'assurer que la reclassification est bien réalisée dans l'onglet Données.
