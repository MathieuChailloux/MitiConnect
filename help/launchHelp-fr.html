Lancements


Cet onglet permet de gérer les lancements pour chaque couple scenario / espêce.
Pour chaque couple, il faut lancer les traitements dans l'odre suivant :
- Landuse : crée la couche d'occupation du sol (reclassifiée and fusionnée)
- Friction : crée la couche de friction
- Project : initialise le projet Graphab avec les patchs
- Linkset : crée le jeu de lien (chemins entre les patchs)
- Graph : créer le graphe

Une fois que le graphe est créé, le bouton Spatialize permet d'afficher les valeurs de la métrique locale pour chaque noeud.
