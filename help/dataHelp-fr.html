Dans cet onglet, il faut importer les données pour les rastériser et standardiser, puis définir les couches d'occupation du sol de base qui seront utilisées dans les prochaines étape.
Les imports doivent être effectués (appuyer sur le bouton) avant de définir les couches d'occupation du sol.
La sélection consiste à extraire et uniformiser les données d'entrée. Une sélection s'effectue depuis une couche d'entrée selon un mode et affecte un groupe et une classe au résultat.
