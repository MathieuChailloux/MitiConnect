Données


Dans cet onglet, il faut importer les données pour les rastériser et les standardiser. Les imports doivent être paramétrés puis lancés avec le bouton Importer les données.
Il est ensuite possible de fusionner les imports pour définir de nouvelles couches d'occupation du sol qui seront utilisées dans les prochaines étape.
La table de droite montre les classes générées par les imports et qui seront utilisées dans les prochaines étapes.
