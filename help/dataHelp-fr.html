Données


Dans cet onglet, il faut importer les données pour les rastériser et les standardiser. Les imports doivent être paramétrés puis lancés avec le bouton Importer les données.
Si la couche d'occupation du sol importée doit être complété par d'autres données (infrastructures linéaires, données thématiques, etc.), il est possible de créer une nouvelle couche d'occupation du sol dans l'interface par fusion des imports réalisés (en mettant par exemple une couche de route au-dessus d'une occupation du sol).
La table de droite (Classes) montre les règles de classification générées par les imports ou les scenarios et qui seront utilisées dans les prochaines étapes. Les nouvelles valeurs correspondent aux codes de l'onglet friction.
