Lancements


Cet onglet permet de gérer les lancements pour chaque triplet scenario / esp√®ce / emprise.
Pour chaque triplet, il faut lancer les traitements dans l'odre suivant :
1 - Occupation du sol : crée la couche d'occupation du sol (reclassifiée and fusionnée)
2 - Friction : crée la couche de friction
3 - Projet : initialise le projet Graphab avec les patchs
4 - Jeu de liens : crée le jeu de liens (chemins entre les patchs)
5 - Graphe : créer le graphe

Une fois que le graphe est créé, le bouton 6 - Spatialiser permet d'afficher les valeurs de la métrique locale pour chaque noeud.

Le bouton 7 - Comparer les scenarios permet de calculer la métrique globale pour chaque scenario et espêce sélectionnés, puis de les afficher sous forme de graphique dans une nouvelle fenêtre.
